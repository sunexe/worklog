# 工作日志管理系统

一个简单易用的工作日志管理工具，帮助您轻松管理和生成工作报告。

## 功能特点

- 📅 日历式界面，直观选择日期
- 📝 分类记录每日工作内容
  - 基础性工作
  - 增值性工作
  - 提升性工作
  - 紧急性工作
- 📊 选择起止日期生成标准格式工作报告
- 🌓 支持明暗主题切换
- 💾 本地数据存储，保护隐私

## 安装和使用

1. 克隆仓库
```bash
git clone https://github.com/你的用户名/工作日志管理系统.git
```

2. 使用本地服务器运行（例如 VS Code 的 Live Server 插件）

3. 在浏览器中访问即可使用

## 技术栈

- HTML5
- CSS3
- JavaScript
- IndexedDB
