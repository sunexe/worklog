一个简单易用的工作日志管理工具，帮助您轻松管理和生成工作周报。

功能特点

- 📅 日历式界面，直观选择日期
- 📝 分类记录每日工作内容
  - 基础性工作
  - 增值性工作
  - 提升性工作
  - 紧急性工作
- 📊 生成标准格式工作报告
- 🌓 支持明暗主题切换
- 💾 本地数据存储，保护隐私

使用方法

1. 点击日历中的日期，输入当天的工作内容
2. 工作内容分为四类：
   - 基础性工作：常规项目交付管理实施进展情况
   - 增值性工作：创新、复盘沉淀、新商机等
   - 提升性工作：学习新技术、新能力，考取证书
   - 紧急性工作：临时紧急任务
3. 选择起止日期，点击生成报告，可生成标准格式的工作报告
4. 点击导出，可另存为文本文件

本地开发

1. 克隆仓库

2. 使用本地服务器运行（例如 VS Code 的 Live Server 插件）

3. 在浏览器中访问
